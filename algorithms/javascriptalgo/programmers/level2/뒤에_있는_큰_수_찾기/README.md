## 뒤에 있는 큰 수 찾기
#### 1. 문제 파악
- 정수로 이루어진 배열 numbers가 있습니다. 
- 배열의 각 원소들에 대해 자신보다 뒤에 있는 숫자 중에서 자신보다 크면서 가장 가까이 있는 수를 뒷 큰수라고 합니다.
- 정수 배열 numbers가 매개변수로 주어질 때, 모든 원소에 대한 뒷 큰수들을 차례로 담은 배열을 return 하도록 solution 함수를 완성해주세요. 
- 단, 뒷 큰수가 존재하지 않는 원소는 -1을 담습니다.
- 제한 사항
  - 4 ≤ numbers의 길이 ≤ 1,000,000
  - 1 ≤ numbers[i] ≤ 1,000,000

#### 3. 계획 세우기
- 정답 answer 배열을 `numbers`의 길이만큼 만들고 -1로 채운다.
- 인덱스를 저장하는 스택을 할당한다.
- for 문을 돌면서 확인한다.
  - while 문을 사용하여 현재 숫자보다 작은 숫자를 체크한다. 
  - 스택이 비어있지 않고, 현재 숫자보다 스택의 마지막 인덱스의 숫자가 크다면
  - 스택에서 pop 하고 해당 인덱스를 `answer[인덱스]`에 현재 숫자로 지정한다.
  - while 문을 빠져나오면 스택에 인덱스를 push 한다.
- answer 를 리턴한다.
