#### 1. 문제 파악
- 주어진 2차원 배열 맵(m x n)에서 1은 육지, 0은 물로 표현되어 있다.
- 섬의 갯수를 구하고 반환한다.

#### 3. 계획 세우기
- 완전탐색을 통하여 1로 연결된 섬인지 아닌지 판단하는 섬 체크 함수가 필요하다. 이 체크 함수는 재귀함수로 탐색한다.
  - 상, 하, 좌, 우를 탐색하여 맨 끝부분을 넘었는지 체크
  - 1이 아닌 0인지 체크
  - 탐색이 끝났다면 해당 요소는 0으로 처리한다.
- 이러한 체크가 모두 통과했다면 카운터를 추가한다.
