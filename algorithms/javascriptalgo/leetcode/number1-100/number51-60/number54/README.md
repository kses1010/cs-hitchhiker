- 변수는 top, bot, left, right 로 설정
- 오른쪽
  - `matrix[top][i]` -> `top++`
- 아래
  - `matrix[i][right]` -> `right--`
- 왼쪽
  - `top <= left` 일 경우 실시: 그렇지 않으면 정사각형이 아닌 직사각형일 경우 한번 더 실행한다.
  - `matrix[bot][i]` -> `bot--`
- 위
  - `left <= right` 일 경우 실시: 그렇지 않으면 정사각형이 아닌 직사각형일 경우 한번 더 실행한다.
  - `matrix[i][left]` -> `left++`
