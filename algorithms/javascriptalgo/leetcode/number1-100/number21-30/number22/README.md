#### 3. 계획 세우기
- 재귀함수를 사용하여 열린 괄호, 닫힌 괄호, n 을 비교한다.
- 만약 열린 괄호의 수와 닫힌 괄호의 수가 같고 열린 괄호의 갯수가 n과 같다면
  - 해당 결과물(`bracket`)을 `res`에 push 하고 리턴한다.
- 만약 열린 괄호가 n 보다 작다면
  - 열린 괄호 + 1, 결과물(`bracket + "("`) 과 함께 그대로 재귀
- 만약 닫힌 괄호가 열린 괄호 수보다 작다면
  - 닫힌 괄호 + 1, 결과물(`bracket + ")"`)과 함께 그대로 재귀
- 답변을 담아내는 `answer`에 빈배열을 할당한다.
- 재귀 함수를 완성했다면 초기화 하여 해당 재귀함수를 작동한다. (`0, 0, n, answer, ""`)
- `answer`을 리턴한다.
